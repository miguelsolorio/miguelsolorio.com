AI-first notebooks bring prompting, code generation, and data workflows into one surface so people can move from an idea to a runnable analysis faster.

# Focus areas

- Prompt-to-cell creation with clear, editable output
- Inline suggestions that preserve notebook structure and readability
- Trust and transparency patterns for generated code

# Outcome

The project established a direction for making notebooks feel collaborative and agentic without sacrificing control for expert users.
